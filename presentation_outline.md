## Project Overview

Smart Study Partner is a simple, local-first study assistant with a small web UI and optional Python tooling.

---

## Key Features

- Local web UI (`index.html`) for quick study sessions
- Lightweight styling in `style.css`
- Optional Python script (`smart_study.py`) for automation or extra features

---

## Requirements

- Python 3.8+ (for the generator and `smart_study.py` features)
- A modern web browser to open `index.html`

---

## Usage

1. Open `index.html` in your browser to use the UI.
2. Run `smart_study.py` for optional Python-backed features.

---

## Development

- Edit `index.html` and `style.css` to change the UI
- Extend `smart_study.py` for backend behaviour
- Run the presentation generator to produce slide deck

---

## Future Directions

- Add note export/import (JSON/Markdown)
- Local search and tagging
- Optional plugin API for spaced-repetition
